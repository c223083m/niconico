Webデザインと3Dモデリングを勉強中。ニコニコ動画が好きです。

自己紹介

こんにちは、みづこです。
大学のゼミでWebデザインやアプリケーション開発、3Dモデリングなどを学んできました。 特に、ユーザーが触って「楽しい」「面白い」と感じられるようなインタラクティブな表現や、世界観のあるデザインを作るのが好きです。

このポートフォリオサイトも、私が大好きな2010年代のニコニコ動画の雰囲気をリスペクトして、遊び心を持って作りました。

趣味・好きなこと

  • 動画視聴: ニコニコ動画でゲーム実況やボカロ曲を巡回するのが日課です。
  • ゲーム制作: Unityを使って簡単なアクションゲームを作るのにハマっています。
  • 3Dモデリング: Blenderでオリジナルキャラクターを作ったり、ワールドを作ったりしています。
  • 散歩: アイディアに行き詰まったら近所を散歩してリフレッシュします。

保有スキル

Webフロントエンド HTML5, CSS3, JavaScript (ES6+), React, Next.js, Tailwind CSS
3D制作・ゲームエンジン Blender (モデリング/リギング), Unity (C#)
デザインツール Figma, Adobe Photoshop, Illustrator
その他 Git/GitHub, Firebase, Vercel
